Queries

This is Query

todo

This will return specific todo-list with inputted todoID
Returns a Todo!

Name Description
todoID - Int!

Example

Query
query todo($todoID: Int!) {
  todo(todoID: $todoID) {
    ID
    Title
    Description
    Condition
  }
}
Variables
{"todoID": 987}
Response 
{
  "data": {
    "todo": {
      "ID": 987,
      "Title": "abc123",
      "Description": "xyz789",
      "Condition": false
    }
  }
}
Queries

todos

This will return all todo-list data
Returns [Todo!]!

Example

Query
query todos {
  todos {
    ID
    Title
    Description
    Condition
  }
}
Response 
{
  "data": {
    "todos": [
      {
        "ID": 123,
        "Title": "xyz789",
        "Description": "xyz789",
        "Condition": false
      }
    ]
  }
}

Types

Boolean

The Boolean scalar type represents true or false.

Example
false
Types

Int

The Int scalar type represents non-fractional signed whole numeric values. Int can represent values between -(2^31) and 2^31 - 1.

Example
987
Types

String

The String scalar type represents textual data, represented as UTF-8 character sequences. The String type is most often used by GraphQL to represent free-form human-readable text.

Types

Todo

This is Todo type

Field Name Description
ID - Int! ID should be integer
Title - String! Todo Title and should be string
Description - String! Description explain about todo-list Title and should be string
Condition - Boolean! Condition check the todo-list done or undone
Example
{"ID": 987, "Title": "abc123", "Description": "xyz789", "Condition": true}
Types

TodoInput

This is Todo Input

Input Field Description
Title - String!

Todo-list Title
Description - String!

Todo-list Description
Condition - Boolean!

Todo-list condition which is done or undone
Example
{"Title": "abc123", "Description": "abc123", "Condition": false}
